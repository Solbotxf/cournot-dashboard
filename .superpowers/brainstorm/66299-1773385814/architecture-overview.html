Key Design Decisions

Frontend Only Changes

All new code lives in this Next.js repo. Backend gets API spec to implement independently.

Auth Token in Headers

JWT passed via Authorization: Bearer header through the existing proxy route.

Role-Based Route Guard

Admin layout component checks role before rendering. Non-admins see access denied.

Reuse Existing Pipeline

PoR trigger calls the same /step/resolve. Resolution confirmation uses a new dedicated endpoint.
